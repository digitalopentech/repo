Para criar uma tabela no Apache Pinot 1.1 e inserir dados nela, siga estes passos detalhados:

1. Preparar o Esquema
Crie um arquivo chamado schema.json com o seguinte conteúdo:

json
Copiar código
{
  "schemaName": "exampleSchema",
  "dimensionFieldSpecs": [
    {
      "name": "customerId",
      "dataType": "STRING"
    },
    {
      "name": "orderId",
      "dataType": "STRING"
    }
  ],
  "metricFieldSpecs": [
    {
      "name": "amount",
      "dataType": "DOUBLE"
    }
  ],
  "dateTimeFieldSpecs": [
    {
      "name": "orderDate",
      "dataType": "LONG",
      "format": "1:MILLISECONDS:EPOCH",
      "granularity": "1:MILLISECONDS"
    }
  ]
}
2. Preparar a Configuração da Tabela
Crie um arquivo chamado tableConfig.json com o seguinte conteúdo:

json
Copiar código
{
  "tableName": "exampleSchema_OFFLINE",
  "tableType": "OFFLINE",
  "segmentsConfig": {
    "timeColumnName": "orderDate",
    "schemaName": "exampleSchema",
    "replication": "1"
  },
  "tenants": {
    "broker": "DefaultTenant",
    "server": "DefaultTenant"
  },
  "tableIndexConfig": {
    "loadMode": "MMAP"
  },
  "ingestionConfig": {
    "batchIngestionConfig": {
      "segmentIngestionType": "APPEND",
      "segmentIngestionFrequency": "DAILY"
    }
  },
  "metadata": {}
}
3. Adicionar o Esquema e a Configuração da Tabela
Use os comandos do pinot-admin.sh para adicionar o esquema e a configuração da tabela ao Apache Pinot.

Adicionar o Esquema:

sh
Copiar código
./pinot-admin.sh AddSchema -schemaFile /path/to/schema.json -controllerHost localhost -controllerPort 9000 -exec
Adicionar a Configuração da Tabela:

sh
Copiar código
./pinot-admin.sh AddTable -tableConfigFile /path/to/tableConfig.json -schemaFile /path/to/schema.json -controllerHost localhost -controllerPort 9000 -exec
4. Preparar os Dados
Crie um arquivo CSV com os dados que você deseja inserir, por exemplo, data.csv:

csv
Copiar código
customerId,orderId,amount,orderDate
123,abc,100.5,1625011200000
124,def,200.75,1625097600000
125,ghi,300.25,1625184000000
5. Criar o Segmento
Use o comando para criar um segmento a partir do arquivo CSV:

sh
Copiar código
./pinot-admin.sh CreateSegment \
  -dataDir /path/to/data \
  -format CSV \
  -outDir /path/to/output \
  -tableConfigFile /path/to/tableConfig.json \
  -schemaFile /path/to/schema.json \
  -overwrite
6. Carregar o Segmento
Depois de criar o segmento, carregue-o no Apache Pinot:

sh
Copiar código
curl -X POST -F 'file=@/path/to/output/exampleSchema_OFFLINE_0.tar.gz' "http://localhost:9000/segments?tableName=exampleSchema_OFFLINE"
Resumo dos Passos
Crie o arquivo schema.json com a definição do esquema.
Crie o arquivo tableConfig.json com a configuração da tabela como OFFLINE.
Adicione o esquema e a configuração da tabela usando os comandos pinot-admin.sh.
Crie o arquivo CSV (data.csv) com os dados.
Crie o segmento usando o comando pinot-admin.sh CreateSegment.
Carregue o segmento criado usando uma solicitação HTTP.
Certifique-se de substituir /path/to/ pelos caminhos corretos no seu sistema. Se precisar de mais assistência, estou à disposição!
